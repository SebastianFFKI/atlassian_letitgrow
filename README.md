# LetItGrow - Intelligente Gartenhelfer-App

## 🌿 Projektübersicht
Mobile App (iOS & Android) zur Unterstützung bei der Gartenpflege mit KI-gestützten Funktionen und Community-Features.

## 📋 Jira-Integration
- **Projekt:** LET - LetItGrow
- **Confluence:** [LetItGrow Documentation](https://mycareernow.atlassian.net/wiki/spaces/LE/overview)
- **Jira Board:** [LetItGrow Tasks](https://mycareernow.atlassian.net/jira/software/projects/LET/boards)

## 🎯 Kernfunktionen
- Pflanzenpflege-Planer mit Erinnerungen
- KI-basierte Pflanzen- und Krankheitserkennung
- Community-Features und Chat
- Smarthome-Integration
- Standortbasierte Empfehlungen
